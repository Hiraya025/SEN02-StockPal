## 🚀 The Vision
Small and medium-sized business (SMB) owners in retail and manufacturing often spend their nights stressing over manual inventory tracking, lost stock, and spreadsheet errors. This constant anxiety takes away from the joy of running their business. 

**StockPal** is a smart, lightweight inventory management MVP designed to give business owners their peace of mind back. By automating real-time tracking and providing visual low-stock alerts, StockPal removes the logistical headaches of inventory management so owners can focus on doing what makes them happy.

## ✨ Core Features (MVP)
* **Real-Time Dashboard:** A clean, responsive interface for immediate supply chain visibility.
* **Instant Stock-In/Out:** Rapid logging of new inventory items and stock levels.
* **Automated Low-Stock Alerts:** The system actively monitors threshold limits and visually flags items requiring restocking.
* **Frictionless CRUD Operations:** Add, view, and delete stock records instantly without page reloads.

## 🛠️ Tech Stack
This project was built for speed, reliability, and seamless cloud deployment during the 24-hour hackathon sprint.
* **Backend:** Python + Flask (Deployed via Vercel Serverless Functions)
* **Database:** Vercel Postgres (Powered by Neon)
* **Frontend:** HTML5, Vanilla JavaScript, Tailwind CSS (via CDN)
* **Deployment:** Vercel
